运行微信机器人有多种协议可以使用，有以下三种不同的协议方式, 每个协议都有不用的特点，如果你不知道选什么，建议使用Padlocal协议来运行你的第一个微信机器人。

- [使用Padlocal协议(推荐使用)](./use_padlocal_getting_started.md)
- 使用Paimon协议
- 使用免费Web协议(目前已不可用)
- 使用puppet-xp协议（仅windows下可用）
